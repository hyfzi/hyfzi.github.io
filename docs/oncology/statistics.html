1.1 客观缓解率及各组率的95%CI

1.1.1 确切概率法（Clopper-Pearson）法

方法简介

Clopper-Pearson法（确切概率法）是一种基于二项分布的精确置信区间估计方法，常用于估计二分类终点（如ORR、DCR等）的率及其95%置信区间。适用于样本量较小的情况。

SAS模版代码

proc freq data=orr;
by trt01pn;
weight count/zeros;
tables resp/binomial(exact level='1') alpha=0.05;
ods output binomialcls=cl_orr;
run;

其中，数据集orr的格式为:

trt01pn resp count
1 0 5
1 1 6
2 0 7
2 1 0

其中，TRT01PN为治疗组别；RESP为响应变量，RESP=1代表达到客观缓解，RESP=0代表未达到客观缓解；COUNT为每个TRT01PN*RESP类别下的例数。需要主要的是，即使某个类别下例数为0，我们也要在数据集中展示。

参数说明

参数 说明
binomial 二项分布
exact 采样exact法计算置信区间
level='1' 基于第1水平（将达到客观缓解作为分子）计算二项分布率
binomialcls 输出基于二项分布的精确置信区间

输出结果

输出结果通常包括：

  • Proportion：响应率（Response Rate）
  • LowerCL：95%置信区间下限
  • UpperCL：95%置信区间上限

1.2 率差及其95%置信区间

1.2.1 Miettinen-Nurminen法

一种基于score统计量、用于两独立样本率差置信区间估计的稳健方法，在小样本或极端事件率情况下较Wald法具有更好的统计性能。

ods trace on;
ods output PdiffCLs=PdiffCLs;
proc freq data=orr;
  weight count/zeros;
  table trt01pn*resp/riskdiff(cl=mn column=2)  alpha=0.05;
run;
ods output close;
ods trace off;

输入数据集orr同上。

参数说明

参数 说明
riskdiff 计算两组间率差（Risk Difference）
cl=mn 采用MN法估计率差的置信区间
column=2

基于第二列事件率（达到客观缓解）计算组间率差（resp=1）。

默认是第一行-第二行（即trt01pn=1 减去 trt01pn=2，试验组-对照组）
PdiffCLs 输出率差及其置信区间结果的数据集

输出结果

  • RiskDifference：率差（客观缓解率，试验组-对照组）

  • LowerCL：95%置信区间下限

  • UpperCL：95%置信区间上限

1.2.2 Cochran-Mantel-Haenszel（CMH）法-分层率差

CMH法常用于随机分层临床试验中二分类终点的组间比较。该方法通过在各分层内计算组间率差，并采用Mantel-Haenszel方法进行加权合并，从而得到考虑分层因素后的共同率差（Common Risk Difference）及其95%置信区间。

相比直接计算总体率差，CMH法能够降低分层因素不平衡对结果的影响，因此广泛应用于ORR、DCR等疗效终点分析。

ods trace on;
ods output CommonPdiff=CommonPdiff;
proc freq data=orr_stratirt;
  weight count/zeros;
  table stratirt*trt01pn*resp/commonriskdiff(cl=mh column=2)  alpha=0.05;
run;
ods output close;
ods trace off;

输入数据集orr_stratirt的格式为：

stratirt trt01pn resp count
男 1 0 3
男 1 1 4
女 2 0 0
女 2 1 3

其中，stratirt代表分层因素变量，可以存在多个，如果存在多个分层变量，可通过strat1 * strat2 * strat3 * trt01pn * resp。需要主要的是，数据集的格式必须为分层变量 * 治疗组别 * 应答变量。因为我们会column=2作为事件率，行1-行2计算率差，所以输入表格的最后两列必须为 trt01p和resp。

参数说明

参数 说明
stratirt 分层因素
commonriskdiff 计算考虑分层因素后的共同率差（Common Risk Difference）
cl=mh 采用Mantel-Haenszel方法估计率差的95%置信区间
column=2 基于第2列事件率计算组间率差，通常对应响应率
CommonPdiff 输出共同率差及其95%置信区间结果的数据集

输出结果

  • Value：公共率差

  • LowerCL：95%置信区间下限

  • UpperCL：95%置信区间上限

2 无进展生存期相关统计方法

  1. 采用Kaplan-Meier法估计各治疗组的中位PFS及其95%CI，并构建各治疗组的Kaplan-Meier曲线。

  2. 采用Kaplan-Meier法估计在3、6、9、12个月（或其他合适时间点）时的无进展生存率及其95% CI。

  3. 采用对数秩检验（log-rank test）进行组间比较。

2.1 Kaplan-Meier法估计各治疗组的中位PFS及其95%CI

/*未分层*/
proc lifetest data=adtte outsurv=occur1 method=KM timelist=3,6,9,12 reduceout;
  time aval*cnsr(1);
    strata trt01pn / test=logrank;
    ods output homtests=homtests1 Homstats=stats1 quartiles=quar1;
run;

/*分层计算logrank*/
proc lifetest data=adtte outsurv=occur1 method=KM;;
  time aval*cnsr(1);
  strata stratirt1 stratirt2 stratirt3 / group= trt01pn test=logrank;
  ods output homtests=homtests1 Homstats=stats1;
run;

问题：为什么计算logrank或HR值，有分层和未分层两种方式，但计算中位无进展生存期或各时点无进展生存率，就不会考虑分层和未分层？

答：“估计性描述”和”组间比较”是两类不同的统计问题。中位PFS和PFS率是在“描述组本身”，而logrank或HR是组间比较。
