实体瘤

1 RECISTv1.1影像评估标准的相关定义

Response Evaluation Criteria in Solid Tumors (RECIST) 1.1是实体瘤疗效评估的标准方法，主要用于评估肿瘤对治疗的反应。RECIST 1.1定义了肿瘤的测量方法、评估时间点以及不同疗效类别的标准。

1.1 RECISTv1.1病灶

在RECISTv1.1评估标准中，病灶定义为：与研究适应症直接相关的病灶，包括：

  • 任意大小非淋巴结恶性病灶

  • 最短垂直径≥10mm的淋巴结病灶

  • 腹水、胸腔或心包积液

将参与者肿瘤负荷的病灶分为可测量及不可测量两类。在肿瘤疗效时点评估中，则根据靶病灶、非靶病灶和新病灶的状态进行总体疗效评估。即：每个时间点都会有1.靶病灶评估，2.非靶病灶评估，3.新病灶评估，4.总体疗效评估。其中，总体疗效评估是综合靶病灶、非靶病灶和新病灶做出的综合疗效判读。程序员也将会直接用4.总体疗效评估去衍生BOR。

1.2 可测量病灶

在断层扫描图像层厚≤5mm的影像检查中，可测量病灶定义为可以在连续两个层面上显示，且轴向断层平面上进行精确测量的病灶，并且其尺寸需要满足以下条件：

  • 非淋巴结病灶：最长径（LDi）≥10mm

  • 淋巴结病灶：最长垂直径（SDi）≥15mm

注意：如果层厚>5mm，则可测量病灶的最小值为层厚的两倍。

1.3 不可测量病灶

不可测量病灶包括：

  • 小病灶，非淋巴结病灶，即最长径（LDi）＜10mm的非淋巴结病灶，以及10mm≤最长垂直径（SDi）＜15mm的淋巴结；

  • 真正不可测量病灶。

1.4 靶病灶

靶病灶为可测量病灶的子集，病灶总数不能超过5个，且每个器官不能超过2个。

1.5 非靶病灶

所有不可测量病灶或符合靶病灶，但因数量限制而未能归为靶病灶的可测量病灶

1.6 新病灶

与基线相比新出现，且符合1.1 RECISTv1.1病灶的定义。

1.7 总结

RECISIT v1.1适用于能形成实体肿块的肿瘤（病灶），血液肿瘤不能用RECISIT，是因为白血病、淋巴瘤血里是肿瘤细胞，不能那尺子量。而前列腺癌为什么又部分适合，是因为前列腺癌属于实体瘤，但是很多转移到骨头里，骨病灶不容易按RECIST测量。

实体瘤评估的具体过程：
1. 基线评估：在治疗开始前，对患者的肿瘤进行全面评估，选定一定数量的靶病灶和非靶病灶，计算所有靶病灶的直径和。对应CRF中的1.肿瘤评估_靶病灶（筛选期），2.肿瘤评估_非靶病灶（筛选期）两个页面。
2. 肿瘤评估：在治疗过程中，按照预定的时间点对患者进行评估，如靶病灶的径长、直径和，非靶病灶的状态以及是否出现新病灶等。对应CRF中的1.肿瘤评估_靶病灶，2.肿瘤评估_非靶病灶，3.肿瘤评估_新病灶三个页面。
3. 总体评估：研究人员为每个时间点的靶病灶、非靶病灶、新病灶的肿瘤评估结果进行分类，同时结合所有病灶判断总的总体评估结果。后续说的总体评估结果都是指结合所有病灶得到的总体评估结果。对应CRF中的1.肿瘤评估_总体疗效一个页面。
4. 最佳总体疗效（BOR）：根据3所有病灶总的总体评估结果，计算整个治疗过程中患者达到的最佳疗效类别，作为最终的疗效评估结果。CRF不收集BOR，通常由编程衍生。

2 总体疗效（Time point response）

2.1 总体疗效评估介绍

最佳时点疗效由CRF中肿瘤评估_总体疗效页面直接收集，不需要衍生计算。

  1. 靶病灶评估
    1. 基线无靶病灶
    2. 完全缓解(CR)：所有靶病灶消失，淋巴结短径<10mm。
    3. 部分缓解(PR)：以基线为参照，靶病灶直径总和至少减少30%。
    4. 疾病进展(PD)：以既往直径总和最小值作为参照，靶病灶直径总和至少增加20%，且增加的绝对值至少5mm。
    5. 疾病稳定(SD)：既不满足PR也不满足PD的情况。
    6. 无法评估(NE)：无法评估靶病灶的情况。
  2. 非靶病灶评估
    1. 基线无非靶病灶
    2. 完全缓解(CR)：所有非靶病灶消失，且肿瘤标志物正常，淋巴结短径<10mm。
    3. 疾病进展(PD)：出现一个或多个新病灶，或非靶病灶明显恶化。
    4. 非完全缓解/非疾病进展(Non-CR/Non-PD)：既不满足CR也不满足PD的情况。
    5. 不可评估(NE)：无法评估非靶病灶的情况。
  3. 新病灶评估
    1. 有新病灶
    2. 无新病灶
  4. 总体评估（所有病灶总的总体评估结果）
    1. 总体评估结果分两类，一类是基线存在靶病灶，另一类是基线无靶病灶。两种情况的总体评估判断规则如下表所示。

2.2 总体评估日期（RS.RSDTC）

总体疗效评估日期需要由程序员衍生，因为EDC中收集的是研究者给出的判定日期，而分析需要的是肿瘤的影像学检查日期。

总体评估日期（Date of Response）始终基于影像学检查日期，而非CRF总体评估页面中收集的评估日期（研究者给出的判定日期）。如果针对特定访视的影像学检查在不同日期进行，若总体评估为疾病进展（PD），则总体评估日期为该访视内最早一次的影像学检查日期；否则为最晚的影像学检查日期。后续提及的肿瘤评估日期或总体评估日期，都是说的这个日期。

3 CRF

4 实体瘤疗效终点

4.1 分类

4.1.1 最佳总体疗效(Best overall response, BOR)

最佳总体疗效（BOR）为基线后、疾病进展或新抗前最佳的总体评估结果。通常CRF不会直接收集，由编程计算。最佳总体疗效可分未确认的和未确认的两类。

4.1.1.1 确认的最佳总体疗效

  • 确认的最佳总体疗效判断规则如下：
    • CR：在新抗或疾病进展前存在至少两次评估结果为CR，且两次评估之间至少间隔4周。
    • PR：在新抗或疾病进展前存在至少两次评估结果为PR或更好（PR-CR,PR-PR），且两次评估之间至少间隔4周。PR-SD-PR能被确认为PR，若两个PR之间的间隔大于等于4周。
    • SD：基线后起至少X周后、疾病进展及新的抗肿瘤治疗开始前，至少有一次疗效评估为稳定或更优，但未被确认为CR、PR。X通常设为6周。
    • Non-CR/Non-PD：基线后起至少X周后、疾病进展及新的抗肿瘤治疗开始前，至少有一次疗效评估为非完全缓解/非疾病进展或更优，但未被确认为CR、PR。X通常设为6周。(Non-CR/Non-PD仅在基线无靶病灶的情况下存在，且后续分析也可能合并为SD)
    • PD：基线后新抗前存在疗效评估为PD，且未被确认为CR、PR或SD。
    • NE：除上述情况外的其他情况。NE的原因需要被汇总记录。
      • 所有基线后肿瘤总体评估均为NE。
      • 基线后首次评估前开启新的抗肿瘤治疗。
      • 无基线后肿瘤评估。
      • SD持续时间不满足≥42天。
      • 其他
Note

值得注意的是，CR后面不能跟PR、SD。PR后面能跟一个SD，如PR-SD-PR可以被确认为PR。但PR-SD-SD-PR不能被确认为PR。若中间存在小于等于两个NE,这可被确认为PR，如PR-NR-PR,PR-NE-NE-PR,PR-SD-NE-PR,PR-NE-SD-PR等都可以被确认为PR。

4.1.1.2 未确认的最佳总体疗效

  • 未确认的最佳总体疗效判断规则如下：
    • uCR：在新抗或疾病进展前存在CR。
    • uPR：在新抗或疾病进展前存在PR。
    • SD：基线后起至少X周后、疾病进展及新的抗肿瘤治疗开始前，至少有一次疗效评估为稳定或更优，但未被确认为uCR、uPR。X通常设为6周。
    • Non-CR/Non-PD：基线后起至少X周后、疾病进展及新的抗肿瘤治疗开始前，至少有一次疗效评估为稳定或更优，但未被确认为uCR、uPR。X通常设为6周。(Non-CR/Non-PD仅在基线无靶病灶的情况下存在，且后续分析也可能合并为SD)
    • PD：基线后新抗前存在疗效评估为PD，且未被确认为uCR、uPR或SD。
    • NE：除上述情况外的其他情况。NE的原因需要被汇总记录。
      • 所有基线后肿瘤总体评估均为NE。
      • 基线后首次评估前开启新的抗肿瘤治疗。
      • 无基线后肿瘤评估。
      • SD持续时间不满足≥42天。
      • 其他

4.1.2 客观缓解率(overall response rate, ORR)

客观缓解率是指在特定分析集中BOR为CR或PR的患者比例。也分为基于确认的BOR和未确认的BOR计算的两个版本的ORR。

4.1.3 疾病控制率(disease control rate, DCR)

疾病控制率是指在特定分析集中BOR为CR、PR或SD的患者比例。也分为基于确认的BOR和未确认的BOR计算的两个版本的DCR。

4.1.4 临床获益率(clinical benefit rate, CBR)

临床获益率是指在特定分析集中BOR为CR、PR或SD持续至少T周的患者比例。（T通常设为12周）。也分为基于确认的BOR和未确认的BOR计算的两个版本的CBR。
